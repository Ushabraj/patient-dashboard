Patient Dashboard UI
📌 Project Overview
This project is a Patient Dashboard UI built by converting an Adobe XD design into a functional web interface using HTML, CSS, and JavaScript. The dashboard displays patient information, diagnosis history, vital signs, and lab results in a structured and responsive layout.


🚀 Features
Responsive patient dashboard layout
Header navigation bar
Sidebar with patient list and profile images
Diagnosis history chart using Chart.js
Vital signs cards (Respiratory Rate, Temperature, Heart Rate)
Diagnostic list table
Patient details panel
Lab results section
Clean and structured UI


🛠️ Technologies Used
HTML5
CSS3 (Flexbox & Responsive Design)
JavaScript
Chart.js
